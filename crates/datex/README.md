# DATEX

> This crate contains the DATEX library for native platforms, as well as the DATEX Command Line Interface (CLI) for interacting with the DATEX runtime.

[![](https://img.shields.io/crates/v/datex.svg)](https://crates.io/crates/datex) [![Twitter badge][]][Twitter link] [![Discord badge][]][Discord link]

The DATEX library provides native implementations for various DATEX functionalities, including communication interfaces. The DATEX CLI offers a REPL (Read-Eval-Print Loop) for executing DATEX code interactively and a command to run DATEX files.

## Installation
The DATEX CLI can be installed on various platforms. Below are the installation methods for different operating systems.

You can install DATEX CLI using Cargo:
```bash
cargo install datex
```
